# PlanetaryDefence

## Description

This is a game where you have to defend the earth from asteroids. You can shoot them with your laser cannon. You can also use the laser cannon to destroy the asteroids before they hit the earth. If you destroy an asteroid you get points. If an asteroid hits the earth you lose points.

## Challenges
This game was built as part of a challenge to build a game as fast as possible

`Time Taken: 2 hours 57 minutes`

## Controls
Use the `A` , `D` keys to move the laser cannon left and right. Use the `S` key to use powerup.
